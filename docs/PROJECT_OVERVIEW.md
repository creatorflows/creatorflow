# Project Overview

## What is CreatorFlow?

CreatorFlow is a collaboration tool for social media teams to manage content creation.

## Problem We Solve

Teams use 5+ different tools (Trello, Drive, WhatsApp, Email) causing confusion.

## Our Solution

One platform for:

1. Task management
2. File sharing with versioning
3. Feedback on specific timestamps
4. Team coordination

## Target Users

- Small social media teams (2-10 people)
- Influencers working with editors
- Local brands creating content

## Tech Stack

- **Frontend**: React + TypeScript
- **Backend**: Node.js + Express
- **Database**: PostgreSQL
- **Storage**: Cloudflare R2 (like S3)
